## 📣 Annonce de la Mise à Jour Graphique : Modale d'Intervention

Bonjour à tous,

Nous avons mis à jour l'apparence de la fenêtre **Modifier l'Intervention** pour améliorer la lisibilité, le contraste et l'identification rapide des données clés.

### 1. ⚙️ Spécifications pour le Conteneur du Modal :

Le conteneur principal du modal doit avoir les styles suivants pour améliorer le contraste visuel :

* **Fond du Modal (Blanc Cassé) :**
    ```css
    background-color: #F7F7F7; /* Gris très clair */
    ```
* **Ombre Portée (Profondeur) :**
    ```css
    box-shadow: 0 8px 25px rgba(0, 0, 0, 0.1); /* Ombre douce */
    border-radius: 8px; /* Coins légèrement arrondis */
    ```

### 3. 🎨 Champs Colorisés :

Veuillez appliquer les codes couleurs définis dans le CRM (fonds très légèrement saturés) aux champs **Agence**, **Attribué à** et **Type Métier** pour faciliter le repérage visuel.
